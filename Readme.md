# 各个文档的说明
### 众包

+ 《众包》
  - 众包概念的宏观讲解，第5页提到众包的三类参与者。 
+ 《众包综述》
  - 第三节关于众包的模式，第四节关于众包的几个应用场景，可以看一下。
+ 《广义的众包-共享经济和众包商业模式》
  -  广义的众包场景。
+ 《Reward or Penalty - Aligning Incentives of Stakeholders in Crowdsourcing》
   - 这是我之前的一篇论文，讲的是中心化众包平台里对worker的奖惩机制，算是本系统的初期工作。而本系统加入了对边缘云的奖励机制，以及众包业务流程设计，以实现去中心化的众包。
   - 论文摘要：众包平台的三个参与者即任务发布者、工人和平台之间存在一定的利益冲突，因而会对众包任务的完成质量、花费、时延和平台发展产生不利影响。 从整个众包社区的长期良性发展的角度入手，设计了一种机制以统一三者的利益关系。一方面，根据工人上报的任务结果及其对该报告的主观把握程度以决定对其实施奖励或惩罚及其奖惩程度，并进一步得到最终答案给任务发布者。另一方面，每个工人的历史表现被设计用来影响其奖惩程度，而同时能够确保奖惩的激励相容性。理论分析和仿真实验结果验证了所设计机制的有效性。

+ 《比特币原理》
+ 《比特币脚本及交易分析 - 智能合约雏形》
  - 硬货
